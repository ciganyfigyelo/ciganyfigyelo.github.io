How to disable the MacOS startup sound temporarily when using OpenCore Legacy Patcher

24 November, 2024

I made this tutorial because nowhere to be found any article about it.

What is a startup sound?

The startup sound or chime is a short sound effect that is played after the user presses the power button. Apple introduced the MacOS startup chime back in 1984 and has evolved over the years.

How to disable the startup sound?

The option to disable or enable in didn't came until MacOS 11 (Big Sur). Since that you can disable it in the System Preferences > Sound > Play sound on startup.

When using Dortania's OpenCore Legacy Patcher this option is not working as expected.

How to disable it when using OCLP?

You can't fully disable it but using a command in Terminal can mute the startup sound temporarily. Write the following code into your Terminal: sudo nvram SystemAudioVolume=%80. This command sets the startup sound volume to zero.
